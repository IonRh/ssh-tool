# ssh_tool
## 一键脚本，修改root用户默认登录验证为秘钥验证，以及一键修改root用户密码
🎉 ssh_tool 修改秘钥登录,修改root密码
```
wget -N https://raw.githubusercontent.com/kwxos/ssh-tool/main/ssh_tool.sh && chmod +x ssh_tool.sh && ./ssh_tool.sh
```
